# Zircon Engine

A multiplatform, open source game engine for 2D and 3D games.

## Features

- **2D & 3D Support** - Unified engine supporting both 2D and 3D game development
- **Pure C** - Written entirely in C for maximum portability and control
- **Multiplatform** - Designed to run on multiple platforms
